📌 常见问题（FAQ）

🔹 如何管理或取消订阅？

您可以通过 [iOS 设置 → Apple ID → 订阅] 来管理或取消您的订阅。

🔹 如何导入本地印章素材？

在应用内，进入【印章管理】页面，点击【导入】按钮，选择本地相册图片，即可导入并使用。

🔹 如何删除印章作品/保存印章作品到相册？

在应用内，进入【印章管理】页面，点击页面底部【删除/保存】按钮，勾选印章，点击【删除】或【保存到相册】按钮，即可完成操作。

🔹 遇到崩溃或闪退怎么办？

请尝试以下方法：

  • 关闭并重新打开应用
  • 检查设备存储空间是否充足
  • 卸载并重新安装应用
